# 🏨 Sistema de Gestión de Hotel

Este es un proyecto para la gestion de un hotel, que se hizo para facilitar el control de las reservas, los clientes, las habitaciones y tener una mejor administración general. El objeto de este proyecto es mejorar la administracion y que sean mas faciles las operaciones que se hacen dia a dia.
---

## 🚀 Funciones principales
- Registro de habitaciones.
- Cambiar estado de la habitación (Disponible-Ocupado-Limpieza).
- Ver Habitaciones
- Eliminar habitaciones.
